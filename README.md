# Nova tabs

This package will help you to create tabs in your resource detail & form page.

## Requirements

- PHP (^8.1 or higher)
- Laravel Nova (^4.12 or higher)

## Installation

Require the package with composer

```bash
composer require shuvroroy/nova-tabs
```
